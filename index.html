File Transfer Protocol Server

This FTP server written in C establishes a listening socket and handles client connections using pthreads. It is capable of handling client commands such as changing the current directory using an operating system call, user authentication, and file transfer using a secondary passive connection strictly for data transfer.

File Transfer Protocol Client

An FTP client written in Java that establishes a connection to a FTP server. It accepts command line instructions from the user and submits the appropriate FTP commands to the server to facilitate file transfer.
